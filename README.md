# alfred-tor

> Alfred 3 workflow - Toggle Tor and Internet proxy on/off


## Install

```
$ npm install --global alfred-tor
```

*Requires Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `tor`, <kbd>Enter</kbd>.
